# Fuchsia WLAN Countries, Jurisdiction, Regulatory Library


* Codes are hosted at `//garnet/lib/rust/wlan-reg`. 
* File `jiri_manifest` contains the `jiri` manifest used to create this repository.
* The final form will be in library, while its intermediate form can also have executable binaries. 
* Sandbox repo `sandbox/porce/reg` will contain exploratory ideas and codes.
* None of the code here shall not be considered as part of Fuchsia code base.
* None of the code here shall not be used for software defined radio's operation.
